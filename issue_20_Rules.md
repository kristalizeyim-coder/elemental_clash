# Issue #20: Kural Dokümantasyonu

### Puanlama Sistemi
- **Predator:** İki kartın toplamı kadar puan kazandırır.
- **Joker:** Rakip kart + 15 puan kazandırır.
- **Sinerji:** Echo koleksiyonunda aynı element varsa x1.5 çarpan uygulanır.
- **Ceza:** Oyun sonunda elde kalan kartların değeri toplam skordan düşülür.
